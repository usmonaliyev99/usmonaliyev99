# My packages

I build few packages for laravel.

## redium

A high-performance laravel auth package powered by redis, inspired by sanctum.

<!-- termynal -->

```bash
$ composer require usmonaliyev/redium
```

To learn more and how to use this package, please consult the [official documentation](https://github.com/usmonaliyev99/redium).

This package provides simple usage of rabbitmq.

```bash
$ composer require usmonaliyev/laravel-simple-rabbitmq
```

If you want to know see how to use it, go to [github repository](https://github.com/usmonaliyev99/laravel-simple-rabbitmq)
