# Main cisco CLI modes

1. User EXEC Mode

```bash
Router>
Switch>
```

Purpose:

- Basic monitoring only
- No configuration allowed

2. Privileged EXEC Mode

```bash
Router#
Switch#
```

Enter with:

```
enable

```

Purpose:

- Full monitoring access
- Can enter configuration modes

3. Global configuration mode

Prompt:

```bash
Router(config)#
Switch(config)#
```

Enter with:

```bash
configure terminal

```

Purpose:

- Configure the entire device
